# Описание:

Разработка всех проектов ведется локально, 
с помощью технологий docker compose
и фиксируется в GitLab. 
После чего с помощью CI\CD, код отправляется в открытый канал на GitHub
а так же формируется билд фронт приложений на vds, 
где поднята прод среда на docker compose, nginx proxy manager.
